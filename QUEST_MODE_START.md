## 📋 Как начать работу

### Шаг 0: Изучи документацию

Перед написанием кода **прочитай:**

4. [CONTENT_STRUCTURE.md](./CONTENT_STRUCTURE.md) — что писать

---

### Шаг 1: Настрой проект

Ты уже в папке `method-club/`.

**Установи зависимости:**
```bash
npm install
```

**Настрой env:**
```bash
cp .env.example .env.local
```

Отредактируй `.env.local` — вставь реальные ссылки (или оставь заглушки).

**Создай базовые папки и файлы:**
```bash
mkdir -p src/app src/components/ui src/sections src/lib src/styles public/fonts public/images
touch src/app/layout.tsx src/app/page.tsx src/styles/globals.css
```

**Запусти dev-сервер:**
```bash
npm run dev
```

Открой http://localhost:3000 — должна быть пустая страница (это ок).

---

### Шаг 2: Настрой дизайн-токены

**a) `tailwind.config.js`**  
Уже создан. Проверь, что все токены на месте (цвета, шрифты, радиусы).

**b) `src/styles/globals.css`**  
Создай файл и добавь:
- `@tailwind` директивы
- `@font-face` для Steelfish
- Глобальные стили для `body`, заголовков
- Правило для `prefers-reduced-motion`

Код есть в [DESIGN_SYSTEM.md](./DESIGN_SYSTEM.md) → раздел "Глобальные стили".

**c) `src/app/layout.tsx`**  
Подключи:
- Inter через Google Fonts (в `<head>`)
- `globals.css`
- Базовую разметку (`<html>`, `<body>`)

---

### Шаг 3: Создай базовые компоненты

**Файлы:**
- `src/components/ui/Container.tsx`
- `src/components/ui/Button.tsx`
- `src/components/ui/Card.tsx`

Код есть в [DESIGN_SYSTEM.md](./DESIGN_SYSTEM.md) → раздел "Компоненты".

**Создай также:**
- `src/lib/utils.ts` — функция `cn()` для слияния классов

```tsx
import { clsx, type ClassValue } from 'clsx';
import { twMerge } from 'tailwind-merge';

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

---

### Шаг 4: Проверь, что всё работает

На этом этапе у тебя должно быть:
- ✅ Проект запускается (`npm run dev`)
- ✅ Tailwind работает (можешь добавить тестовый `<div className="text-gold-2">Test</div>`)
- ✅ Базовые компоненты импортируются без ошибок

**Покажи результат:**  
Скриншот страницы + структуру файлов.

**Жди фидбека** — не переходи к Hero, пока не получишь OK.
