# МЕТОД:КЛУБ — лендинг (Next.js + Tailwind)

Этот репозиторий — кодовая версия лендинга **МЕТОД:КЛУБ** (страница клуба), перенесённая с Tilda на Next.js + React + TypeScript + Tailwind CSS.

Главная цель — сохранить все смыслы и тексты действующей страницы клуба, но:
- сделать структуру понятнее и более модульной,
- усилить подачу и визуальные акценты,
- добавить несколько фич из референса [milanatarba.ru/evolution2](https://milanatarba.ru/evolution2),
- сделать сайт живым и премиальным, без ощущения типичного «инфобиза».

- **Текущий сайт клуба:**  
  https://agkedu.ru/method_club?utm_source=op&utm_medium=2025&utm_campaign=club

  - Тёмный премиальный фон (`#040404`)

  - Золотой градиент для акцентов и кнопок
  - Стеклянные карточки (glass effect: прозрачный белый фон, blur, лёгкие тени)

## Ключевые принципы

### 1. Контент не выдумывать
Все обещания, результаты, формулировки — только из текущей страницы клуба. Можно переформатировать, усиливать подачу, но не придумывать новых «результатов» и «гарантий».

### 2. Стилистику сохранить
- Ощущение премиального тёмного сайта с золотыми акцентами.
- Стеклянные карточки, глубина, воздух, аккуратные подсветки.
- Никаких чужих визуальных языков (не «другой бренд»).

### 3. Работа строго блок за блоком
Никакого «напалма» по всей странице. Каждый блок:
- спроектировать,
- согласовать,
- реализовать и проверить,
- только после этого переходить к следующему.

### 4. Обязательные фичи
- **Блок с телефоном + бот** (mockup iPhone + скриншоты бота, слайдер).
- **Стоимость в день:**
  - Месяц: 5000 ₽ → **167 ₽/день**
  - Год: 50 000 ₽ → **137 ₽/день**
- **5 типов анимаций** + поддержка `prefers-reduced-motion`.
- **Опциональный таймер** до ближайшего эфира (если задана дата).
- **Telegram** службы заботы.
- **Юридический футер** с реквизитами.

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Настройка переменных окружения
cp .env.example .env.local
# Отредактируй .env.local с реальными ссылками

# Запуск dev-сервера
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.
